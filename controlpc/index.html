Control Pc -beta

Control Pc es una herramienta que está pensada para controlar una laptop desde un dispositivo móvil, debido a que muchos profesores emplean diapositivas, imágenes o videos de un ordenador en sus clases y debido a que no podemos pasar las diapositivas a distancia a medida que nos movemos por el aula(Tenemos que pasar las diapositivas manualmente) cree esta herramienta, aunque aún está en desarrollo.

Requisitos.

Tener un ordenador windows con bluetooth

Desventajas

Debido a la complejidad del sistema operativo Windows y a problemas de compatibilidad con los distintos dispositivos y controladores es posible que esta opción no funcione para todos, y aun si funciona puede ser complejo utilizarla.

¿Cómo puedo controlar mi pc mediante bluetooth?

En tu dispositivo Android.

Para hacer uso de esta herramienta solo necesitas presionar en el botón "Start" en el apartado "Control PC" de la aplicación móvil, si el bluetooth del dispositivo no está encendido, la aplicación solicitará permiso para activarlo, debes aceptar, aparecerá un icono que indica que esperes.

En tu ordenador.

  1. Deberás descargar la versión de Control PC par windows desde aquí.
  2. Empareja tu dispositivo Android al ordenador mediante bluetooth, si no sabes como hacerlo, sigue esta guía.
  3. En la aplicación Control PC para windows presiona el botón ON, debería aparecer una lista de dispositivos, elige el teléfono con el que quieras controlar el ordenador.
  4. Después de un tiempo debería desaparecer el icono de espera en la aplicación Android y aparecer una pantalla similar al trackpad de una laptop, felicidades, ahora puedes controlar el ordenador mediante esta herramienta.

Recuerda que esta herramienta está aún en desarrollo, es posible que encuentres errores, en la mayoría de los casos deberás apagar el bluetooth de ambos dispositivos y volverlo a encender, esto debería solucionar la mayoría de problemas que pueda presentar.
